飞镖杀杀
========
原谅我取这么一个名字...我确实不知道改取什么名字...所以...

这是一个demo，基于libgdx，主要使用了libgdx的2d ui组件。

记住，这只是一个demo，所有资料均来自网络。
如果用于其他用途，请谨慎。

Libgdx是基于Apache Licence 2.0协议的，详细信息可以去[libgdx官网](http://libgdx.badlogicgames.com/)看看。
